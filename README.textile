h1. In Place Rich Editor

AJAX In Place Rich Editor: In place AJAX-powered WYSIWYG editor

Using the great libraries Prototype 1.7, Script.Aculo.Us 1.9.0 and TinyMCE 3.4.2, IPRE provide a really easy way to implement an in place AJAX-powered editor using WYSIWYG.

h3. Tested on

Firefox 3+, Internet Explorer 6+, Safari 4+, Chrome 11+ and Opera 10+
